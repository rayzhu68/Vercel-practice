# Vercel Practice Project

This is a tiny monorepo-shaped deployment test.

It has:

- `web`: a minimal Next.js frontend
- `api`: a placeholder backend folder

Use it to practise the same Vercel setting needed by EninTree:

```text
Repository root: vercel-practice
Vercel Root Directory: web
Framework Preset: Next.js
Install Command: npm install
Build Command: npm run build
Output Directory: blank/default
```

After deployment, the Vercel URL should show:

```text
Hello from Vercel practice
```

If this works, the Vercel account and GitHub connection are fine.
If EninTree still shows 404 after this works, the remaining issue is specific to the EninTree Vercel project settings.
